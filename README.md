# 📅 GetSchedule (gs) — Парсер расписания ФГОУ ПСК

Консольная утилита для быстрого получения расписания занятий прямо в терминал. Больше не нужно заходить в браузер и вводить логин — достаточно одной команды.

---

## 🛠 Стек технологий

* **Python 3.10+** — основной язык разработки.
* **Requests** — взаимодействие с веб-сервером (авторизация и сессии).
* **BeautifulSoup4** — парсинг HTML-контента.
* **Rich** — создание красивого интерфейса, таблиц и индикаторов загрузки в терминале.
* **Argparse** — обработка аргументов командной строки.

---

## 🚀 Установка и настройка

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python. Затем установите необходимые библиотеки:
```bash
pip install requests beautifulsoup4 rich

```

### 2. Создание команды `gs`

Чтобы скрипт вызывался короткой командой `gs` из любой папки:

**Для Windows (PowerShell):**

1. Откройте профиль: `notepad $PROFILE`
2. Вставьте строку:
`function gs { python "C:\путь\к\файлу\gs.py" $args }`
3. Перезапустите терминал.

**Для Linux / macOS:**

1. Откройте конфиг: `nano ~/.bashrc` (или `~/.zshrc`)
2. Добавьте алиас:
`alias gs='python3 /путь/к/файлу/gs.py'`
3. Примените: `source ~/.bashrc`

---

## 📖 Инструкция по использованию

### 🔑 Первый запуск (Авторизация)

Для начала нужно один раз сохранить ваши данные (они запишутся в локальный файл `config.json`):

```bash
gs --s -l ВАШ_ID -p ВАШ_ПАРОЛЬ

```

* `-l` (или `--login`): ваш ID студента.
* `-p` (или `--password`): ваш пароль.

### 📅 Просмотр расписания

После настройки используйте следующие флаги:

* **На сегодня:**
```bash
gs -td

```


* **На завтра:**
```bash
gs -to

```

## ⚠️ Возможные ошибки

* **"Расписание еще не опубликовано"** — если на сайте нет данных на выбранный день.
* **"Ошибка авторизации"** — если вы ввели неверный ID или пароль при настройке. Повторите команду `gs --s -l ... -p ...` с правильными данными.
